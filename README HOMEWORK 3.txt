ВТОРОЕ ЗАДАНИЕ
У меня определен список фильмов, которые не были просмотрены.
Я прохожусь по этому списку фильмов и высчитываю вероятность того, когда был просмотрен данный фильм: 
в будный день или в выходной. Если вероятность просмотра в будний день выше, то фильм добавляем в итоговый массив.

Выбор фильма для просмотра осуществляется следующим образом:
Проходимся по всем фильмам из первого задания (они уже отсортированы по оценкам): если фильм есть в результативном
массиве второго задания, его высчитанная оценка больше средней оценки самого пользователя и т.к. нам нужен только один фильм, 
отслеживаем его находимость с помощью флага flagEmpty (изначально True - значит фильм еще не найден).
Если же такой фильм не был найден и флаг остается равным True, то ранее для такого результата
был создан массив additionalMovies
(т.к. было найден пять похожих пользователей на моего одного, я взяла у каждого первый самый высоко оцененный фильм 
и добавила в additionalMovies). 
И потом предлагаю пользователю посмотреть любой из этих фильмов (выбирается случайным образом).
